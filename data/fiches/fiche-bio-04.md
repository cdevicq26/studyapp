## 1. Acides nucléiques & chimie du vivant (Ch3)

### Le nucléotide et la double hélice

Un **nucléotide d'ADN** = 1 phosphate (P) + 1 sucre désoxyribose (S) + 1 base azotée (A, T, C ou G).

L'ADN est un **polymère de nucléotides** organisé en **double hélice** :
- 2 brins **antiparallèles** (l'un va de 5'→3', l'autre de 3'→5')
- Les bases s'apparient par liaisons hydrogène : **A=T (2 liaisons H)** et **C≡G (3 liaisons H)**
- Règle de Chargaff : %A = %T et %G = %C (ces pourcentages varient selon l'espèce)

> 💡 **À retenir :** AT ≠ TA — l'ordre compte car les brins sont antiparallèles.

### ADN vs ARN

| Propriété | ADN | ARN |
|---|---|---|
| Structure | Bicaténaire (double brin) | **Monocaténaire** (simple brin) |
| Bases | A, T, G, C | A, **U** (Uracile), G, C |
| Sucre | Désoxyribose | **Ribose** |
| Durée de vie | Longue | Plus courte |
| Fonction | Stockage de l'information | Transport, transcription, catalyse |

**3 types d'ARN :** ARNm (messager — code pour les protéines), ARNt (transport — porte les AA), ARNr (ribosomal — constitue les ribosomes).

### Niveaux de condensation de l'ADN

| Niveau | Description |
|---|---|
| **1. Double hélice** | 2 nm — liaisons H entre bases |
| **2. Nucléosome** | hélice enroulée autour d'histones → "collier de perles" (10 nm) |
| **3. Chromatine** | nucléosomes enroulés — état standard de l'ADN dans la cellule |
| **4. Chromosome** | stade ultime, visible seulement pendant la division |

- **Hétérochromatine** = foncée, gènes inactifs (ne s'expriment pas)
- **Euchromatine** = claire, gènes actifs (s'expriment)
- **Chromosome bichromatidien** = 2 chromatides reliées par le **centromère** ; **télomères** = extrémités protectrices

### Le dogme fondamental

```
ADN  →(Transcription, dans le noyau)→  ARNm  →(Traduction, au ribosome)→  Protéine
```

**Transcription** : l'**ARN polymérase** lit le brin matrice de l'ADN (sens 5'→3' sur l'ARN produit) et crée un ARNm complémentaire (transcrit primaire). Se déroule dans le **noyau**.

**2 modifications post-transcriptionnelles (eucaryotes) :**
| Modification | Description | Rôle |
|---|---|---|
| **Coiffe 5'** | Guanine modifiée + 3 phosphates | ↑ durée de vie, export du noyau, reconnaissance ribosome |
| **Queue poly-A (3')** | 100-250 adénines | ↑ durée de vie, export, reconnaissance ribosome |

**Épissage :** retire les **introns** (non-codants), conserve les **exons** (codants) → ARNm mature. L'**épissage alternatif** permet à un seul gène de produire plusieurs protéines différentes (ex. gène CGRP → calcitonine OU CGRP).

### La traduction au ribosome

- **Codon** = triplet de nucléotides sur l'ARNm (ex. AUG = START = Méthionine)
- **Anticodon** = séquence complémentaire portée par l'ARNt
- **ARNt** porte un acide aminé et reconnaît un codon via son anticodon
- **64 codons** = 61 codants + 3 STOP (UAA, UAG, UGA)

**Sites du ribosome :**
- **Site A** (Accepteur) : reçoit le nouvel ARNt
- **Site P** (Peptide) : formation de la liaison peptidique
- **Site E** (Évacuation) : éjecte l'ARNt vide

**5 étapes :**
1. **Initiation** : le ribosome glisse sur l'ARNm jusqu'au codon START (AUG) au site A
2. **Arrivée du 1er ARNt** : ARNt-Méthionine (anticodon complémentaire à AUG) au site A
3. **1re élongation** : le ribosome avance, l'ARNt-Met passe au site P, un nouvel ARNt arrive en A → liaison peptidique
4. **Suite de l'élongation** : avance d'un codon, ARNt vide passe au site E et est éjecté, nouvel ARNt en A, nouvelle liaison peptidique — répétition
5. **Terminaison** : le site A atteint un codon STOP → libération de l'ARNm et de la protéine

> 💡 **Schéma à savoir décrire** : l'ARNm défile dans le ribosome ; à chaque "pas" un ARNt apporte un AA en site A, une liaison peptidique se forme en site P, et l'ARNt vide repart par le site E. La chaîne d'AA s'allonge à chaque pas jusqu'au codon STOP.

**Localisation des ribosomes :**
- **Libres** dans le cytosol → protéines qui restent dans la cellule
- **Accrochés au RER** → protéines exportées hors de la cellule

## 2. Les mutations (Ch4)

> **Mutation** = modification aléatoire d'une séquence d'ADN.

### Les 3 grandes catégories

| Type | Définition | Exemple |
|---|---|---|
| **Ponctuelles** | 1 nucléotide modifié | Drépanocytose (GAG→GTG) |
| **Chromosomiques** | 1 morceau de chromosome modifié | Translocation, délétion |
| **Génomiques** | Nombre de chromosomes modifié | Trisomie 21 |

### Mutations ponctuelles (sur séquence ATCGTTCAG)

| Type | Résultat | Explication |
|---|---|---|
| **Substitution** | ATCCTTCAG | 1 base remplacée |
| **Addition** | ATCGATTCAG | 1 base ajoutée |
| **Délétion** | ATCG_TCAG | 1 base supprimée |
| **Inversion** | ATCGTCTAG | Séquence inversée |

**Conséquences d'une substitution :**

| Conséquence | Définition | Exemple |
|---|---|---|
| **Faux sens** | Change l'acide aminé → protéine modifiée | GAG→GTG (GLU→VAL) |
| **Non-sens** | Crée un codon STOP prématuré | TTT→ATT (LYS→STOP) |
| **Silencieuse** | Aucun changement d'AA (code dégénéré) | GGG→GGT (PRO→PRO) |

> 💡 **Drépanocytose — exemple de référence :** substitution sur le brin codant, **A→T** au 2e nucléotide du codon (**GAG→GTG**). Le 6e acide aminé de la β-globine passe de **acide glutamique (Glu)** à **valine (Val)** → hémoglobine S → globule rouge en forme de **faucille**. C'est une mutation **faux sens**.

> ⚠️ **Chorée de Huntington :** répétition du triplet **CAG** sur le chromosome 4. ≤35 répétitions = sain, >35 = malade (maladie neurodégénérative apparaissant à l'âge adulte).

### Mutations chromosomiques (5 types)

| Type | Description |
|---|---|
| **Délétion** | Perte d'un segment chromosomique |
| **Duplication** | Répétition d'un segment |
| **Inversion** | Segment retourné dans le même chromosome |
| **Insertion** | Ajout d'un segment étranger |
| **Translocation** | Échange de segments entre chromosomes non-homologues |

### Mutations génomiques

Modification du **nombre de chromosomes** : **monosomie** (2n−1) / **trisomie** (2n+1).

**Caryotype humain normal** = **46 chromosomes = 22 paires d'autosomes + 1 paire de chromosomes sexuels (XX/XY)**.

| Syndrome | Caryotype | Caractéristiques principales |
|---|---|---|
| **Klinefelter** | XXY | Stérilité, répartition graisses atypique, problèmes pilosité, possible retard mental |
| **Turner** | XO | Retard de développement physique/mental, stérilité, très petite taille |
| **Trisomie 21 (Down)** | 3 copies du chromosome 21 | Problèmes cardiaques, taille réduite, yeux bridés, retard mental |

## 3. La cellule (Ch5)

### Les 4 structures présentes dans TOUTE cellule

> 💡 **ADN / Membrane / Ribosome / Cytosol** — présentes chez les eucaryotes ET les procaryotes.

### Reconnaître les organites au MET

| Organite | Forme reconnaissable | Membrane(s) |
|---|---|---|
| **Mitochondrie** | Allongée (patate), crêtes mitochondriales | 2 |
| **Noyau** | Ovoïde, contient le nucléole | 2 |
| **RER** | Canaux parallèles avec ribosomes (points sombres) autour | 1 |
| **REL** | Tubules lisses sans ribosomes | 1 |
| **Golgi** | Empilement de saccules ("pile d'assiettes") | 1 |
| **Lysosome / Peroxysome** | Petits organites arrondis | 1 |
| **Ribosome** | Petits points sombres, libres ou fixés au RER | 0 |
| **Chloroplaste** | Empilement de grana, ovoïde | 2 |
| **Vacuole centrale** | Grand espace clair/translucide (végétale) | 1 |
| **Centrosome/centriole** | Paire de structures cylindriques perpendiculaires | 0 |

### Tableau comparatif animale / végétale / procaryote

| Critère | Cellule animale | Cellule végétale | Procaryote |
|---|---|---|---|
| Noyau | ✓ | ✓ | ✗ (nucléoïde à la place) |
| Membrane plasmique | ✓ | ✓ | ✓ |
| Ribosomes | ✓ | ✓ | ✓ |
| ADN | ✓ | ✓ | ✓ (circulaire) |
| Mitochondries | ✓ | ✓ | ✗ |
| Chloroplastes | ✗ | ✓ | ✗ |
| Vacuole centrale | ✗ | ✓ | ✗ |
| Paroi cellulosique | ✗ | ✓ | ✗ (autre type de paroi) |
| Lysosomes/Centrosomes/Flagelles | ✓ | ✗ | — |
| Organisation | Multicellulaire | Multicellulaire | Unicellulaire |

**Exclusivités cellule végétale :** **chloroplaste** (photosynthèse) + **vacuole centrale** (stockage/déchets) + **paroi cellulosique** (rigidité) + **plasmodesmes** (canaux entre cellules).

**Absent chez la cellule végétale :** lysosomes, centrosomes/centrioles, flagelles.

### Rôle des organites principaux

| Organite | Rôle |
|---|---|
| **Noyau** | Contient l'ADN, transcription |
| **Nucléole** | Produit les ribosomes |
| **RER** | Synthèse des protéines (avec ribosomes accrochés) |
| **REL** | Synthèse lipides, détoxification |
| **Golgi** | Tri, modification, expédition des protéines |
| **Mitochondrie** | Respiration cellulaire → ATP |
| **Lysosome** | Digestion (hydrolyse des macromolécules) |
| **Chloroplaste** | Photosynthèse |
| **Ribosome** | Fabrication des protéines |

### Trajet d'une protéine sécrétée

**Ribosome (sur RER) → RER → vésicule → Golgi → vésicule → exocytose → extérieur de la cellule.**

Seules les protéines synthétisées par des ribosomes **accrochés au RER** suivent ce trajet d'export ; les ribosomes **libres** dans le cytosol produisent des protéines qui restent dans la cellule.

### Équations métaboliques à connaître

**Respiration cellulaire** (mitochondrie) :
$$C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O + ATP$$
→ le glucose et l'oxygène sont consommés pour produire de l'énergie (ATP), du CO₂ et de l'eau comme déchets.

**Photosynthèse** (chloroplaste) :
$$6CO_2 + 6H_2O + \text{lumière} \rightarrow C_6H_{12}O_6 + 6O_2$$
→ le CO₂, l'eau et la lumière sont utilisés pour produire du glucose (énergie stockée) et de l'oxygène.

> 💡 Ces deux équations sont quasiment inverses l'une de l'autre.

## 4. Le cycle cellulaire (Ch6)

### Les phases du cycle

| Phase | Nom | Description |
|---|---|---|
| **G1** | Première croissance | Augmentation des molécules, organites, taille |
| **S** | Synthèse | Réplication de l'ADN : 100% → 200% |
| **G2** | Deuxième croissance | Vérification et préparation à la division |
| **M** | Mitose | Répartition de l'ADN dans les futures cellules filles (4-5 phases) |
| **C** | Cytocinèse | Séparation physique des 2 cellules filles |
| **G0** | Dormance | Hors cycle — la cellule fonctionne mais ne se divise plus |

> 💡 **Interphase = G1 + S + G2** (préparation). **Mitose = M** (division nucléaire).

### Évolution de la quantité d'ADN au cours du cycle

| Phase | Taux ADN | Ce qui se passe |
|---|---|---|
| G1 | 100% | ADN non répliqué |
| S | 100% → 200% | Réplication en cours (semi-conservative) |
| G2 | 200% | ADN entièrement répliqué |
| M+C | 200% → 100% | Séparation → chaque cellule fille récupère 100% |

> ⚠️ Toutes les cellules filles sont **génétiquement identiques** — la différence entre G1 et G2 est uniquement la **quantité** d'ADN, pas sa nature.

**Réplication semi-conservative :** chaque nouvelle molécule d'ADN bicaténaire = 1 ancien brin (conservé) + 1 nouveau brin (synthétisé).

### Les 5 phases de la mitose (PPMAT)

| Phase | Ce qui se passe |
|---|---|
| **Prophase** | Condensation de l'ADN en chromosomes (X) · disparition de la membrane nucléaire · formation du fuseau mitotique (microtubules) · les centrosomes migrent vers les pôles opposés |
| **Prométaphase** | Attachement des chromosomes aux microtubules via le **kinétochore** |
| **Métaphase** | Les chromosomes s'alignent au centre (**plaque équatoriale**), chromatides reliées au fuseau via le kinétochore |
| **Anaphase** | **Séparation des chromatides sœurs** → migration vers les pôles opposés |
| **Télophase** | **Reformation de la membrane nucléaire** autour de chaque jeu de chromosomes · décondensation (retour à la chromatine) · début de la cytocinèse |

**Cytocinèse :** séparation physique des 2 cellules filles.
- Cellule animale : **anneau contractile** qui pince la cellule
- Cellule végétale : **plaque cellulaire** entre les 2 noyaux

> 💡 **Mémo ordre** : Prophase → Prométaphase → Métaphase → Anaphase → Télophase (P-P-M-A-T).

**Reconnaître une phase au microscope :**
- **Interphase** : chromatine floue et diffuse (non condensée)
- **Métaphase** : chromosomes alignés en ligne au centre
- **Anaphase** : deux groupes de chromatides s'éloignant vers les pôles
- **Télophase** : reformation de noyaux aux pôles

### Contrôle du cycle et cancer

- **Check G1** : vérification des nutriments/ressources
- **Check G2** : vérification de l'intégrité de l'ADN
- **Inhibition de contact** : les cellules normales s'arrêtent de se diviser au contact des voisines — les cellules **cancéreuses échappent** à ce mécanisme

| | Tumeur bénigne | Tumeur maligne |
|---|---|---|
| Volume | Limité | Croît, envahit les tissus voisins |
| Dissémination | Non | Oui — produit des **métastases** |

**Métastase** = parties de la tumeur qui se détachent et colonisent d'autres organes.

**Traitements :** chirurgie, radiothérapie, chimiothérapie, hormonothérapie, immunothérapie, traitement ciblé.

## 5. Transport membranaire (Ch7)

### Structure de la membrane

La membrane plasmique = **bicouche phospholipidique** (~7 nm) avec protéines enchâssées.

- **Phospholipides** : tête hydrophile (aime l'eau) → orientée vers l'extérieur ET l'intérieur de la cellule ; queues hydrophobes (fuient l'eau) → orientées vers l'intérieur de la bicouche
- **Cholestérol** : module la fluidité
- **Protéines intégrales (transmembranaires)** : traversent toute la bicouche — transport, réception, signal
- **Protéines périphériques** : à la surface
- **Glycoprotéines/glycolipides** : reconnaissance cellulaire

> 💡 **Modèle "mosaïque fluide"** : les phospholipides peuvent bouger latéralement, la membrane est molle et peut se "sceller" si perforée.

### Gradients

**Gradient de concentration** = différence de concentration d'une substance entre 2 points.
- **Selon le gradient** : du + concentré vers le − concentré (spontané, sans énergie)
- **Contre le gradient** : du − vers le + concentré (nécessite de l'énergie)

### Tableau récapitulatif des transports

| Type de transport | Direction | Énergie | Protéine ? | Exemples |
|---|---|---|---|---|
| **Diffusion simple** | Selon gradient | Non | Non | O₂, CO₂, H₂O (osmose) |
| **Diffusion facilitée (canal)** | Selon gradient | Non | Oui (canal) | Na⁺, K⁺, Cl⁻ |
| **Diffusion facilitée (pompe)** | Selon gradient | Non | Oui (perméase) | Glucose (GLUT) |
| **Transport actif** | Contre gradient | Oui (ATP) | Oui (pompe) | Pompe Na⁺/K⁺ |
| **Endocytose** | Entrée | Oui | — | Phagocytose, pinocytose |
| **Exocytose** | Sortie | Oui | — | Sécrétion d'insuline |

> 💡 **Passif vs actif** : passif = selon le gradient, sans ATP (diffusion simple, diffusion facilitée, osmose). Actif = contre le gradient, avec ATP (pompe Na/K, endo/exocytose).

### L'osmose

L'osmose = cas particulier de diffusion simple : déplacement de l'**eau** du milieu **hypotonique** (moins concentré) vers le milieu **hypertonique** (plus concentré), via les **aquaporines**.

**Cellule animale (sans paroi) :**

| Milieu | Ce qui se passe | Résultat |
|---|---|---|
| Hypotonique | H₂O entre | **Cellule lysée** (éclate) |
| Isotonique | Équilibre | **Cellule normale** |
| Hypertonique | H₂O sort | **Cellule crénelée** (ratatinée) |

**Cellule végétale (avec paroi) :**

| Milieu | Ce qui se passe | Résultat |
|---|---|---|
| Hypotonique | H₂O entre, pression sur la paroi | **Cellule turgescente** (la paroi résiste, ne peut pas éclater) |
| Isotonique | Équilibre | **Cellule flasque** |
| Hypertonique | H₂O sort, membrane se détache de la paroi | **Cellule plasmolysée** |

> ⚠️ La cellule végétale **ne peut pas éclater** en milieu hypotonique grâce à sa paroi rigide.

### La pompe Na⁺/K⁺

Transport actif qui consomme de l'ATP :

> 💡 **3 Na⁺ sortent / 2 K⁺ entrent / 1 ATP consommé** par cycle — contre les deux gradients de concentration → maintient les gradients essentiels (influx nerveux, contraction musculaire).

### Endocytose et exocytose

- **Endocytose** : la membrane forme une vésicule pour faire entrer de la matière
  - **Phagocytose** : matières solides (ex. macrophages)
  - **Pinocytose** : matières liquides
- **Exocytose** : une vésicule interne fusionne avec la membrane plasmique → sécrétion vers l'extérieur (lien direct avec le Golgi)

## Schémas clés à savoir décrire

1. **Nucléotide ADN** : phosphate + désoxyribose + base azotée (A, T, C ou G)
2. **Double hélice ADN** : 2 brins antiparallèles reliés par liaisons H (A=T, C≡G)
3. **Dogme fondamental** : ADN (noyau) → transcription → ARNm → sort du noyau → traduction au ribosome → protéine
4. **Traduction au ribosome** : ARNm défile, sites E-P-A, ARNt apporte les AA, liaison peptidique, jusqu'au codon STOP
5. **Évolution de l'ADN dans le cycle cellulaire** : 100% (G1) → 100-200% (S) → 200% (G2) → retour à 100% par cellule fille (M+C)
6. **Mitose PPMAT** : Prophase (condensation, fuseau) → Prométaphase (kinétochore) → Métaphase (alignement équatorial) → Anaphase (séparation chromatides) → Télophase (reformation noyaux)
7. **Membrane plasmique** : bicouche phospholipidique, têtes hydrophiles vers l'extérieur, queues hydrophobes au centre, protéines enchâssées
8. **Pompe Na⁺/K⁺** : 3 Na⁺ sortent, 2 K⁺ entrent, 1 ATP consommé, contre les gradients
